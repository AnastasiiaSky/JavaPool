Для работы с несколькими классами нужен classpath. На этом этапе стоит задуматься
об отделении файлов исходного кода от скомпилированных файлов. Чтобы скомпилировать
группу классов, необходимо из главного каталога использовать команду javac с аргументами:
-d {target} - где target - директория в которую будут помещены скомпилированные классы.
----------------------------------------------------------------------------------
Для компиляции программы, находясь в директории ex00 неоходимо выполнить:
javac -d target ImagesToChar/src/java/edu/school21/printer/*/*.java
----------------------------------------------------------------------------------

Для запуска используется команда java а аргументом:
-classpath - флаг, после которого следует указать местоположение скомпилированных
классов. Java будет искать главный класс и все сопутствующие именно в этой директории.
----------------------------------------------------------------------------------
Для запуска программы, находясь в директории ex00 неоходимо выполнить:
java -classpath ./target ImagesToChar/src/java/edu/school21/printer/app/Program 0 . /Users/qylenett/Desktop/projects/Java_Bootcamp.Day04-1/src/ex00/it.bmp
----------------------------------------------------------------------------------

Удаление после запуска
----------------------------------------------------------------------------------
rm -rf target
----------------------------------------------------------------------------------
