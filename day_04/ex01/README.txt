Чтобы программу было легко переносить и запускать, можно собрать скомпилированные
классы в jar-файл — архив классов. Главное отличие от zip или rar-архивов — наличие
файла манифеста. В этом манифесте указывается главный класс, который будет запускаться
при выполнении jar-файла, classpath, а также много дополнительной информации.
main-class указывает класс, который содержит метод main и будет выполнен при запуске.
class-path — путь к скомпилированным классам или дополнительным библиотекам.
-cmf — флаг, после которого следует указать путь к файлу манифеста.
-С — флаг, после которого указывается путь к скомпилированным классам.
. — путь, куда будет помещен jar-файл.
Теперь можно запустить. Запуск jar-файлов выполняется также с помощью команды java,
но следом нужно указать флаг -jar: он говорит о том, что запускается Jar-файл,
а второй аргумент — путь к jar-файлу, включая расширение.

----------------------------------------------------------------------------------
Для компиляции программы, находясь в директории ex01 неоходимо выполнить:
javac -d target ImagesToChar/java/edu/school21/printer/*/*.java
----------------------------------------------------------------------------------
----------------------------------------------------------------------------------
Для создания архива, находясь в директории ex01 неоходимо выполнить:
jar cmfv ImagesToChar/manifest.txt ./target/app.jar -C target .
----------------------------------------------------------------------------------
----------------------------------------------------------------------------------
Для запуска архива, находясь в директории ex01 неоходимо выполнить:
java -jar target/app.jar . 0
----------------------------------------------------------------------------------
----------------------------------------------------------------------------------
rm -rf target
